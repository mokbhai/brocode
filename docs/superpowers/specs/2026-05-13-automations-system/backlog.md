# Automations Backlog

These ideas are optional and not required for the first complete Automations target.

## Advanced Scheduling

- Full cron editor for expert users.
- RFC 5545 RRULE support.
- Holiday/calendar-aware schedules.
- Per-run randomized jitter.

## Output Destinations

- Post run summaries to Slack, email, or other connectors.
- Create ClickUp/Linear issues from failed runs.
- Export automation reports as documents.

## Templates

- Built-in automation templates for standup summaries, CI triage, release notes, dependency checks, and recurring code health scans.
- User-created template library.

## Multi-Step Workflows

- Chain automations together.
- Conditional follow-up runs based on prior output.
- Human approval gates before write-enabled runs.

## Scheduling Infrastructure

- External scheduler integration.
- Headless/server mode for always-on automations.
- Distributed locking across multiple BroCode instances.

## Result Management

- Automatic result thread rotation by month or run count.
- Run output retention settings.
- Dashboard analytics for success rate, duration, and changed-file volume.
