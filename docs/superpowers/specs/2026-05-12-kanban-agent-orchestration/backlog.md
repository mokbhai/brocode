# Backlog

These items are useful but not required for the complete target in this spec set.

- External tracker sync with GitHub Issues, ClickUp, Linear, or Jira.
- Multiple boards per project.
- Separate worktree per agent attempt with merge/reconcile.
- Automatic task extraction from arbitrary spec text.
- Embedded live transcript rendering inside card detail.
- Automatic PR creation without human approval.
- Cross-card dependency graph.
- Scheduled or recurring card execution.
- Cost budget enforcement beyond visible run counts and limits.
- Provider-specific optimized reviewer modes beyond existing provider capabilities.
